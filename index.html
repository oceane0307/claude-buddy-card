数据架构

CompanionBones (确定性，从不存储)
├── rarity: 'common' | 'uncommon' | 'rare' | 'epic' | 'legendary'
├── species: 18 种物种之一
├── eye: 6 种眼睛样式
├── hat: 8 种帽子
├── shiny: boolean (1% 概率)
└── stats: { DEBUGGING, PATIENCE, CHAOS, WISDOM, SNARK } (1-100)

CompanionSoul (AI 生成，持久化存储)
├── name: string
└── personality: string

Companion = CompanionBones & CompanionSoul & { hatchedAt: number }

属性分配算法

function rollStats(rng, rarity):
  floor = RARITY_FLOOR[rarity]  // common:5, uncommon:15, rare:25, epic:35, legendary:50
  peak  = random pick from 5 stats  // 该属性获得 +50 加成
  dump  = random pick (不同于 peak)  // 该属性被惩罚

  peak stat  = min(100, floor + 50 + rng()*30)   // 高值
  dump stat  = max(1,   floor - 10 + rng()*15)   // 低值
  other stats = floor + rng()*40                  // 中间值

🧬 有趣细节：物种名称编码

// buddy/types.ts
const c = String.fromCharCode
export const duck     = c(0x64,0x75,0x63,0x6b)         as 'duck'

// ... all 18 species encoded this way to bypass excluded-strings.txt
