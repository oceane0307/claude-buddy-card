// BUDDY 宠物系统详解

系统架构

Bones（骨骼 - 确定性数据）

由 hash(userId + SALT) 派生的外观和属性数据，永不存储，每次重新计算。防止用户篡改配置文件。

SALT = 'friend-2026-401'

Soul（灵魂 - AI 生成数据）

首次孵化时由 Claude 生成的名字和性格描述，持久化保存在 config.companion 中。

PRNG 算法：Mulberry32

function mulberry32(seed: number): () => number { let a = seed >>> 0 return function () { a |= 0 a = (a + 0x6d2b79f5) | 0 let t = Math.imul(a ^ (a >>> 15), 1 | a) t = (t + Math.imul(t ^ (t >>> 7), 61 | t)) ^ t return ((t ^ (t >>> 14)) >>> 0) / 4294967296 } } // 种子 = hashString(userId + 'friend-2026-401') // 同一用户永远获得同一只 Buddy

Security
